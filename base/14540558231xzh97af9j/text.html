From the imagemagick package, use the convert command:

convert *.jpg pictures.pdf

convert -compress jpeg *.pnm pictures.pdf

Можно на каждый лист проставить "копия верна":

convert *.jpg -draw 'image SrcOver 200,3000 566,430 "1.png"' out.pdf

(http://www.imagemagick.org/script/command-line-options.php#compress)

1.png должен быть с прозрачным фоном в гимпе: Слой-Прозрачность-Цвет в альфаканал.

You will get a single pdf containing all jpg in the current folder.

Install IM with:

sudo apt-get install imagemagick
