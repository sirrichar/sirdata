Запуск виртуалки без интерфейса только с rdp

Далее подключаемся через remmina (например)

VBoxHeadless --startvm WinXPCodesys --vrde on


Экспорт виртуалок в нужное место из коммандной строки (запускается из нужного места)

VBoxManage export WinXPCodesys test -o ./virtualki_20160601.ova

Смена интерфейса сетевой карты в режиме моста

VBoxManage modifyvm WinXPCodesys --bridgeadapter1 em1

Удаление поддержки USB2.0

VBoxManage modifyvm WinXPCodesys --usbehci off

Информация о машине

VBoxManage showvminfo WinXPCodesys

Управление машиной (pause, resume, poweroff,

reset, savestate)

Подробнее https://www.virtualbox.org/manual/ch08.html#idm3528

VBoxManage controlvm WinXPCodesys poweroff
